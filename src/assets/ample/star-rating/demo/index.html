star-rating.js

This zero-dependency plugin transforms a select with numerical-range values (i.e. 1-5) into a dynamic star rating element.

For convenience, this plugin can also be used as a jQuery plugin (see examples below).

For production, use the files from the dist/ folder.

Usage Example

<link href="css/star-rating.css" rel="stylesheet">

<select id="star-rating">
	<option value="">Select a rating</option>
	<option value="5">Excellent</option>
	<option value="4">Very Good</option>
	<option value="3">Average</option>
	<option value="2">Poor</option>
	<option value="1">Terrible</option>
</select>

<script src="js/star-rating.min.js"></script>
<script>
	var starrating = new StarRating( document.getElementById( 'star-rating' ));
</script>

by default, the images are located in the following relative path from the stylesheet: ../img/star-*.svg. To change this location, either import the SCSS file into your project and change the default SCSS variables, or override the CSS directly in your stylesheet.

Live Examples

Using vanilla javascript

var starrating1 = new StarRating( document.getElementById( 'star-rating-1' ));

Using jQuery

$( '#star-rating-2' ).starrating();

Changing the global plugin defaults

StarRating.defaults.initialText = "Rate me";

var starrating3 = new StarRating( document.getElementById( 'star-rating-3' ));

Using the "data-option" attribute on the <select> to set the options

<select id="star-rating-4" data-options='{"initialText":"Rate this"}'>
	...
<select>

<script>
	var starrating4 = new StarRating( document.getElementById( 'star-rating-4' ));
</script>

Changing all available options in a 7-star rating with a pre-selected <option>

var starrating5 = new StarRating( document.getElementById( 'star-rating-5' ), {
	clearable: false,
	showText: false,
	onClick: function( el ) {
		console.log( 'I selected: ' + el[ el.selectedIndex ].text );
	},
});

Options

Here are the default options

{
    clearable  : true,
    onClick    : null,
    initialText: "Click to Rate",
    showText   : true,
}

clearable

This boolean value determines whether the star rating can be cleared by clicking on the an already pre-selected star.

onClick

This value determines the custom function that is triggered after you click on a star. The custom function argument is the select HTMLElement.

initialText

This string value determines the initial text when no value is selected. This has no effect if showText is set to false.

showText

This boolean value determines whether or not the rating text is shown.

Build

star-rating.js uses gulp to build from src.

$ npm install
$ gulp

The compiled files will be saved in the dist/ folder.
